TBS-16 - Getting issue details... STATUS

  1. Implement the backend API endpoints to fetch booking data and calculate revenue based on the price of each travel package and the number of bookings.

  2. Integrate the frontend with the backend API to request and display the reports on customer bookings and revenue

  3. Design a user-friendly interface to present the reports

  4. Design the database schema to efficiently store and retrieve the booking and revenue data

  5. Validate the calculations of revenue and generated reports

  1. Design a notification system that allows agents to send automated notifications to customers

  2. Integrate a notification service provider, gateway, to send notifications to customers using their provided contact information.

  3. Implement the backend API endpoints to trigger and send notifications based on specific events

  4. Design email templates or message formats for notifications

3.

TBS-18 - Getting issue details... STATUS

  1. Design the agent account management system, including the necessary database tables and relationships to store agent account information.

  3. Test to ensure the reliability and correctness of agent account management operations

  4. Integrate the booking modification functionality into the user-facing interface
