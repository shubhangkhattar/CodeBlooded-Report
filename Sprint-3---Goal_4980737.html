Travel Booking System : Sprint 3 - Goal

Created by Anant Shukla, last modified on Jun 19, 2023

The primary goal of sprint 3 is to provide a range of features focused on facilitating payment processing for bookings, enabling booking cancellations, and handling refunds. Additionally, the aim is to remove outdated packages from the agent's travel booking system to ensure up-to-date information. The sprint also includes UI enhancements to improve the user experience and address minor bugs.
