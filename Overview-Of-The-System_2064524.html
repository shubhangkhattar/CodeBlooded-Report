Travel Booking System : Overview Of The System

Created by Jwalit Chiragbhai Shah, last modified on Jun 01, 2023

System Name

Travel Booking System

Description

The Travel Booking System is an online platform that allows customers to browse and book travel packages. It offers a wide range of options, including pre-made packages as well as individual components such as flights, hotels, and activities. The system handles the entire booking process, from package selection to payment confirmation, and provides a seamless experience for customers. Additionally, it offers features for agents responsible for managing the bookings, allowing them to view, modify, or cancel bookings as needed.

User Goals

  1. Users can join the system as a Travel agent or a Customer and log in to the system.

  2. Users can browse available travel packages. They can also search for travel packages.

  3. Users can select and customize travel packages by selecting individual components.

  4. Users can calculate the total cost.

  5. Users can confirm the booking and submit payment information.

  6. Customers expect to receive a confirmation of their booking once the payment is successfully processed.

  7. Users should also have the ability to modify or cancel bookings if necessary.

  8. Users can update their profile information.
