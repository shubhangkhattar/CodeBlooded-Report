Travel Booking System : Copy of Wireframes for main use cases Sprint-2

Created by Shubhang Khattar on Jun 19, 2023

The Concordia Travel Booking system has been designed with the following proposals in mind

Login Page

The login page includes two input fields designed for existing users (customer & agent) to enter their email and password. Additionally, there are three buttons available: one for accessing the home page, another for signing up as a new user, and a third button for logging in.

Create new user page

The "Create New User" page allows users to create new accounts, whether they are agents or customers. The page includes the following input fields: First Name, Last Name, User Type, Date of Birth (DOB), email, and password. Additionally, there are two buttons available, namely "Login" and "Signup." Furthermore, the page displays a list of all the customers.

Home Screen page

The home screen page presents a range of options tailored to the preferences of the customer or agent. Currently, it offers the ability to create, manage, and search for travel packages. Additionally, there is a convenient logout option available on the page.

Create Booking Page

The "Create Package" page empowers users to generate new bookings by providing essential details such as customer ID, Travel Package ID, and departure date. Additionally, the page conveniently displays a comprehensive list of all the existing bookings available within the system.

Update Booking Page

The updated booking page allows users to modify existing bookings. Users can update details such as the booking id, customer id, travel package ID, departure data and booking status city,. Additionally, the page displays a comprehensive list of all the currently available bookings in the system.

View Booking

Create package page

The "Create Package" page empowers users to generate new packages by providing essential details such as the destination city, destination country, number of days, flight information, activities, hotel options, and price. Additionally, the page conveniently displays a comprehensive list of all the existing packages available within the system.

Update package

The updated package page allows users to modify existing packages. Users can update details such as the destination city, destination country, duration, flight information, activities, hotel options, and price. Additionally, the page displays a comprehensive list of all the currently available packages in the system.

Search travel package page.

The search travel package page enables users to search for existing packages based on the city name.

Update Profile

The Update Profile page enables users to update their profile details such as First Name, Last Name, DOB, Email.

Booking Report by Travel Package Page

Report by Travel Package Page by displays the package details such as destination, number of bookings and total revenue generated.
