TC01

As a customer, I want to create a booking for the selected package, so that I can plan my travel arrangements independently.

  • The user should be already registered

  • The user should be registered as a Customer

  1. Launch the application and log in as a customer

  2. Navigate to the "Create Booking" section.

  3. Verify that the "Create Booking" form is displayed with the following fields:

    • Customer Id

    • Travel Package Id

    • Departure Date

4. Enter valid data into the above fields:

5. Click on the "Create Booking" button

  1. Create Booking page opens successfully.

  2. User is able to enter details in all the fields successfully.

  3. Booking is created successfully.

  4. Created Booking is added to the List of bookings.

Pass
