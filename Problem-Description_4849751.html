Travel Booking System : Problem Description

Created by Anushka Shetty, last modified on Jun 18, 2023

Concordia Travel aims to develop a web application for a travel booking system that allows customers to browse and book travel packages effortlessly. The goal is to provide a user-friendly platform that encompasses flights, hotels, and activities, offering both pre-made packages and the option to create custom packages by selecting individual components. Additionally, customers should be able to create accounts, view their bookings, and make modifications or cancellations as necessary.

To achieve these objectives, several epics have been identified:

  1. Package Management: Creating classes and functionalities for managing travel packages, flights, hotels, and activities. This includes creating, modifying, and searching available packages.

  2. Booking Management: Developing classes and functionalities for managing customer bookings, including customer and payment information. This epic entails creating, modifying, and managing customer bookings effectively.

  3. Agent Management: Establishing classes and functionalities to manage agents responsible for packages and bookings. This epic involves managing agent accounts, generating reports on bookings and revenue, and facilitating communication with customers and agents.

  4. Custom Package Creation: Allowing customers to create personalized travel packages by selecting flights, hotels, and activities. This epic includes developing a user interface for component selection and calculating the total package cost.

  5. Payment Processing: Implementing a payment processing system, potentially by integrating a payment processing API like STRIPE, to handle customer payments securely and efficiently.

  6. User Account Management: Developing classes and functionalities for managing customer and agent accounts, along with implementing robust security measures to safeguard user information.

By addressing these epics, the Concordia Travel company aims to overcome the existing challenges and provide customers with a seamless and user-friendly platform for browsing, booking, and managing their travel packages.
