TC01

As a customer, I want to provide my payment information securely, so that I can complete the booking process.

  • The user should be already registered

  • The user should be registered as a Customer

  • The user should have a supported valid card.

  1. Launch the application and log in as a customer

  2. Navigate to the "Create Booking" section.

  3. Verify that the "Create Booking" form is displayed with the following fields and enter valid data for the fields.

5. Click on the "Provide payment details" button.

6. Enter Card Details.

7. Click on the “Pay“ button

  1. Money is debited from the customer's account.

  2. A payment status message is displayed to the customer.

  3. Booking is created successfully.

  4. Created Booking is added to the List of bookings.

Pass

 

 

TC02

As a customer, I want to receive confirmation of my payment and booking so that I have proof of payment and booking.

  • The user should be already registered

  • The user should be registered as a Customer

  • The payment was successful

  1. User (Customer) should make the successful payment for the booking

  2. Customer should check their email for payment confirmation and successful booking details.

  1. The customer receives an email for a successful payment.

  2. Verify that the list of all created bookings is displayed

Pass

TC03

As an agent, I want to integrate a payment processing API into the system, so that payments can be made by the customer using the application.

  • The agent has an active stripe account.

  • The agent has a valid private and public API keys.

  • The agent has completed all the steps at stripe dashboard required to process payment such as bank details.

  1. The agent needs create a developer account on stripe https://stripe.com/en-ca .

  2. The agent activates the developer API keys.

  3. The agent provides the necessary details at stripe dashboard such as bank details and verification.

  1. Payment integration is successful

  2. Agent successfully receives the payment in their bank account.

Pass

TC04

As a customer, I want the ability to cancel bookings within the system, ensuring my reservations are properly cancelled and refunded if applicable.

  • The user should be already registered

  • The user should be registered as a Customer

  • The user should have a complete booking.

  1. Launch the application and log in as a customer

  2. Navigate to the "View Bookings" section.

  3. Select the booking customer wants to cancel.

  4. Click on cancel booking option.

  1. The booking is successfully cancelled.

  2. The customer receives the refund if applicable.

Pass

TC05

As an agent, I want the capability to cancel bookings on behalf of customers within the system, ensuring their reservations are properly canceled and refunded if applicable.

  • The booking should already be created with completed payment.

  1. Launch the application and log in as a agent

  2. Navigate to the "View Bookings" section.

  3. Select the booking agents wants to cancel.

  4. Click on cancel booking option.

  1. The booking is successfully cancelled.

  2. The customer receives the refund if applicable.

Pass

TC06

As an agent, I need the ability to delete travel packages that were created on the system, allowing me to remove outdated or unavailable travel options and keep the listings up-to-date.

  • The user should be already registered

  • The user should be registered as a Agent

  1. Launch the application and log in as a agent

  2. Navigate to the "View Travel Package" section.

  3. Select the package agents wants to cancel.

  4. Click on delete package option.

  1. The travel package is not visible under “view packages“ to the agent.

  2. The travel package is not visible under “view packages“ to the customer.
